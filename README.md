# DVM Discovery Board
## Intro
The DVM Discovery Board allows students, engineers and DIYers to quickly build and learn key voltage measuring concepts and circuits. Customize 1 to 3 Voltage Ranges. Build the circuit blocks you need. Easily interface to any MicroController (Arduino i this example). Run the available C codem and modify for your project. 

Hands-on experimenting and coding boosts understanding and intuition!
