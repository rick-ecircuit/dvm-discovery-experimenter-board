# DVM Discovery Board
## Intro
The DVM Discovery Board allows students, engineers and DIYers to quickly build and learn key voltage measuring concepts and circuits. Hands-on prototyping and experimenting boosts understanding and intuition!

![dvm pcb, sch](http://www.ecircuitcenter.com/DVM1/discovery/dvm-discovery-top1.PNG)

* Easy-to-use Experimenter Board.
* Circuit Options
	* 1 to 3 Customizable Voltage Ranges
	* Unipolar (+ input only) or Bipolar (+ / - input)
	* Precision Mode (Vref option)
	* Range Select: Manual Push-button or Auto 
* Simple interface to the Arduino UNO (or other MicroController).
* C Code Examples - run, modify, expand!
* Excel Design Calculator - gains, RC values, resolution, settling time.

## Flexible Hands-On Learning
* Quickly build and explore multiple measurement options. 
* Solderless headers for easy component changes.

## Building Blocks
Start with the basic circuit blocks
* Input Divider (2 Ranges)
* Low-Pass Filter
* Push-Button
![Block diagram](http://www.ecircuitcenter.com/DVM1/discovery/dvm-discovery-block1.png)

Add blocks to expand your project
* Amplifier (additional range)
* Negative Rail (bipolar input)
* Voltage Reference (precision mode).

## Micro-Controller & C Code
* Easily interfaces to a MicroController Bd like the Arduino UNO.
* Run the code for the Basic or Advanced Voltmeter. Modify or customize for your project.
	* Basic ( 2-Range, Unipolar Input ):  DVM_discovery_basic_1.ino
	* Advanced ( 3-Range, Bipolar Input ):  DVM_discovery_3range_bipolar_1.ino
